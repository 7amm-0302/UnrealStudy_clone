## 🛠️ 진행 방식
1. 매주 정해진 분량의 강의를 듣는다.
2. 주차별 샘플 코드를 따라 작성하고 주석을 꼼꼼히 작성한다.
3. 배운 내용을 정리한 md 파일도 작성한다.
4. 각자 본인의 GitHub 레포지토리에 업로드 후 Pull Request를 보낸다.
5. 제출 마감은 **모임 시작 전까지**이며, **늦으면 벌금 2,000원** 발생!
